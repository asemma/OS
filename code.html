Code of the robot

The full code of the robot can be found at Github repository.

Structure of SexyBot code

The different parts in the code of the robot are:

  • The main program
  • The sensors
  • The motors
  • The bluetooth interface

Installation

First step: clone the repository

git clone git@github.com:Daudau/OS.git

Second step: push it on the ev3 robot using scp

scp -r OS/ [username]@[robot_ip]:~

Third step: on the robot go into the program directory and compile the code

cd ~/OS
make clean; make install

Last step: run the program

./bin/sexybot
