Code of the robot

Structure of SexyBot code

The different parts in the code of the robot are:

  • The kernel
  • The sensors
  • The motors
  • The bluetooth interface

The kernel

The core of the code of the robot is a function responsible to launch the different thread at the beginning of the programm.<\br> This function will then apply the strategy of the robot while regularly polling the sensors and the bluetooth threads in order to to respond to its environment.

The sensors

Each sensor works in its own thread in order to keep working in the background. They will regularly update the output of the sensor so that the main thread can access this value when needed.
