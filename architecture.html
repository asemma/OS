SexyBot

OS Robot Competition


Architecture of the robot

SexyBot has been built using the Lego MindStorm Ev3 kit.

The actuators

The robot has been built using three motors.

Two of them are used for the wheels. Then, they can be actionned separately.

In order to grab the ball, the plier in front of the robot can go up or down thanks to the third motor.

The sensors

In order to evolve in its environment SexyBot has several sensors.

First of all it has a compass in order to know its orientation. As this compass is not really accurate when the robot is moving, it is only used to make the robot turn precisely when not moving.

SexyBot also use a distance sensor in order to detect the balls, the walls or the obstacles.
