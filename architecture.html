SexyBot

OS Robot Competition


Architecture of the robot

SexyBot has been built using the robot kit ........... sold by MindStorm.

The actuators

The robot has two motor controlled wheels and pliers.

The pliers are made of an arm that can be lowered in order to trap the ball.

The wheels can be moved separately by two motors.

The sensors

In order to evolve in its environment SexyBot has several sensors.

First of all it has a compass in order to know its orientation respectively to the where it started.

SexyBot also use a distance sensor in order to detect the obstacles in front if it.

There is a touch sensor situated in the arm of the robot in order to detect the presence of the ball.
